01Introduction

What is Tkinter?

Tkinter lets you build real desktop apps with windows, buttons, text boxes, and more — all using pure Python. It's already installed with Python, no pip needed.

Why Tkinter?
Built-in→No installation needed — comes with Python
Simple→Beginner-friendly, easy to learn
Cross-platform→Works on Windows, Mac, Linux
Real windows→Creates actual desktop GUI apps
Great for learning→Perfect intro to GUI programming

05Layouts

Placing Widgets — Layouts

After creating a widget you must place it on screen. Tkinter has 3 layout managers — choose the right one for your design.

pack()
Stacks widgets one by one, top to bottom (or left/right).

btn.pack(side="left")
lbl.pack(pady=10)

✅ Best for simple vertical / horizontal layouts.
grid()
Places widgets in rows and columns like a table.

btn.grid(row=0, col=1)
lbl.grid(row=1, col=0)

✅ Best for forms, login screens, calculators.
place()
Places widgets at exact pixel coordinates.

btn.place(x=50, y=100)

⚠️ Harder to maintain — only use when precise positioning is needed.
1# grid() example — simple 2-column form
2tk.Label(window, text="Username:").grid(row=0, column=0, padx=5, pady=5)
3tk.Entry(window).grid(row=0, column=1)
4tk.Label(window, text="Password:").grid(row=1, column=0, padx=5)
5tk.Entry(window, show="*").grid(row=1, column=1)
6tk.Button(window, text="Login").grid(row=2, column=1, pady=8)
06Events

Handling Events

An event is anything the user does — click, key press, mouse move. Use bind() to react to events, or command= on buttons.

Common events
"<Button-1>"→Left mouse click
"<Button-3>"→Right mouse click
"<Return>"→Enter key pressed
"<KeyPress>"→Any key pressed
"<Enter>"→Mouse enters widget
"<Leave>"→Mouse leaves widget
bind() examples
1# Press Enter to submit
2def on_enter(event):
3    name = entry.get()
4    lbl.config(text=f"Hi {name}!")
5 
6entry.bind("<Return>", on_enter)
7 
8# Hover colour change
9btn.bind("<Enter>",
10    lambda e: btn.config(bg="lightblue"))
11btn.bind("<Leave>",
12    lambda e: btn.config(bg="SystemButtonFace"))
⚡ command= vs bind()
Use command=my_func on Button for clicks — it's simpler. Use bind("<event>", handler) for keyboard events or when the handler needs the event object.
07Mini Project

Mini Project — Simple Calculator

Let's put it all together: Entry, Button, Label, grid layout, and event handling — a working calculator!

1import tkinter as tk
2window = tk.Tk()
3window.title("Calculator")
4 
5entry = tk.Entry(window, width=30,
6    font=("Arial", 14), justify="right")
7entry.grid(row=0, column=0,
8    columnspan=4, padx=5, pady=5)
9 
10def press(key):
11    entry.insert(tk.END, key)
12 
13def calculate():
14    result = eval(entry.get())
15    entry.delete(0, tk.END)
16    entry.insert(0, result)
17 
18buttons = [
19    "7","8","9","/",
20    "4","5","6","*",
21    "1","2","3","-",
22    "0",".","=","+"
23]
24for i, btn_text in enumerate(buttons):
25    cmd = calculate if btn_text=="=" else lambda b=btn_text: press(b)
26    tk.Button(window, text=btn_text, width=5,
27        command=cmd).grid(row=i//4+1, column=i%4, padx=2, pady=2)
28window.mainloop()

09Summary

Lesson Summary

Everything you need to build your first Tkinter app

tk.Tk()
Creates the main window. Set title with window.title() and size with window.geometry("400x300").
Widgets
Label, Button, Entry, Text, Checkbutton, Listbox, Frame. Create with tk.WidgetName(parent, ...).
Entry
Text input box. .get() reads it, .delete() clears it, .insert() fills it.
Layouts
pack() for simple stacks. grid(row, column) for forms. place(x, y) for exact position.
Events
command= on buttons. bind("<event>", handler) for keyboard / mouse events.
mainloop()
Always the last line. Keeps the window alive and listens for events. Without it, the window closes immediately.
1import tkinter as tk # import
2win = tk.Tk(); win.title("App") # window
3tk.Label(win, text="Hello").pack() # widget
4tk.Button(win, text="OK", command=win.quit).pack() # event
5win.mainloop() # run ← always last!
