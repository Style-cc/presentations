01for loop

The for loop

Iterates over every element of an iterable — a list, string, tuple, dictionary, or range(). The number of iterations is known upfront.

📋
Known count
Best when you know what you're iterating over — a list, string, or range.
🔤
Any iterable
Works on lists, strings, tuples, dicts, sets, generators — anything iterable.
🔢
Auto variable
Each iteration the loop variable is automatically set to the next element.
📦
Unpacking
Use for k, v in dict.items() to unpack pairs in one go.

02range()

The range() function

range(start, stop, step) generates a sequence of integers on the fly — memory-efficient and the most common way to loop a fixed number of times.

range(5)
0, 1, 2, 3, 4
0 up to 4 — stop is exclusive
range(2, 7)
2, 3, 4, 5, 6
Start from 2, stop before 7
range(0, 10, 2)
0, 2, 4, 6, 8
Step 2 — even numbers only
range(5, 0, -1)
5, 4, 3, 2, 1
Negative step — countdown
1# Sum 1 to 100

03while loop

The while loop

Executes its body as long as a condition remains True. The number of iterations is not known in advance — it depends on data or user input.

Syntax & flow

⚠️ Infinite loop: If the condition never becomes False, the loop runs forever. Always make sure the condition changes inside the body.

06continue

The continue statement

continue skips the rest of the current iteration and jumps to the next one. The loop does not stop — only this step is skipped.

2for n in range(1, 8):
3    if n % 2 == 0:

1# Skip empty lines when reading
2for line in lines:
3    if not line.strip():
4        continue # skip blank lines

break vs continue: break exits the whole loop. continue skips only the current iteration — the loop keeps going.
07else clause

Loop else clause

Python allows an else block on loops. It runs after the loop completes normally — meaning no break was triggered. Useful for "search and not found" patterns.

✅ else runs — no break hit
1for n in [1, 3, 5]:
2    if n % 2 == 0:
3        break
4else:
5    print("No evens found") # ← runs
❌ else skipped — break hit
1for n in [1, 2, 3]:
2    if n % 2 == 0:
3        break # break triggered on 2!
4else:
5    print("No evens") # ← never runs
Practical — linear search
1# Search for a target value
2def find(lst, target):
3    for i, v in enumerate(lst):
4        if v == target:
5            print(f"Found at {i}")
6            break
7    else: # no break hit = not found
8        print("Not in list")
08Nested loops

Nested Loops

A loop inside a loop. Each iteration of the outer loop triggers all iterations of the inner loop. Great for 2D data, matrices, and coordinate pairs.

Multiplication table
1for i in range(1, 4): # outer
2    for j in range(1, 4): # inner
3        print(i * j, end=" ")
4    print() # new line after each row
5# 1 2 3
6# 2 4 6
7# 3 6 9
break in nested loops
1# break only exits the NEAREST loop
2for i in range(3): # outer keeps going
3    for j in range(3):
4        if j == 1: break # inner only
5        print(i, j)
6# 0 0 / 1 0 / 2 0
Key rule: break and continue always apply to the innermost enclosing loop only.
