Dictionaries & Sets

Dictionaries map keys to values for instant lookup. Sets store only unique elements. Both are essential tools in real-world Python code.

{ key: value } O(1) lookup unique keys { unique values } | & - ^ no order
# dict — key → value mapping
person = {"name": "Alice", "age": 25}
print(person["name"]) # "Alice"

# set — unique values only
nums = set([1, 2, 2, 3, 3])
print(nums) # {1, 2, 3}

# set operations
{1,2} | {2,3} # {1, 2, 3} — union

05Set Operations

Set Operations

Sets support mathematical operations: union, intersection, difference, and symmetric difference — all with clean operator syntax.

1A = {1, 2, 3, 4}    B = {3, 4, 5, 6}
🔵🟠
A | B
Union
All elements from both sets
{1, 2, 3, 4, 5, 6}
🔵∩🟠
A & B
Intersection
Only elements in both
{3, 4}
🔵➖🟠
A - B
Difference
In A but not in B
{1, 2}
🔵△🟠
A ^ B
Symmetric Diff
In one, but not both
{1, 2, 5, 6}
1A | B == A.union(B) # union
2A & B == A.intersection(B) # intersection
3A - B == A.difference(B) # difference
4A ^ B == A.symmetric_difference(B) # sym. diff
06Set Methods

Set Methods

Methods for adding, removing elements, and testing relationships between sets.

.add(x)
Add element x
s.add(5) → {1,2,3,5}
.remove(x)
Remove x (raises error if missing)
s.remove(2)
.discard(x)
Remove x (safe — no error)
s.discard(99) # no error
.pop()
Remove & return an arbitrary element
s.pop() # random item
.union(s2)
Return union of two sets
s.union({4,5}) == s | {4,5}
.intersection(s2)
Return intersection
s.intersection({1,9})
.difference(s2)
Return difference
s.difference({1}) == s - {1}
.issubset(s2)
True if s ⊆ s2
{1,2}.issubset({1,2,3}) → True
.issuperset(s2)
True if s ⊇ s2
{1,2,3}.issuperset({1,2})
.isdisjoint(s2)
True if no common elements
{1,2}.isdisjoint({3,4}) → True
