01 Fundamentals

What is a List?

A list stores a collection of items in a specific order. Unlike strings, lists are mutable — you can change their contents after creation.

📋
Ordered
Items preserve their insertion order. lst[0] always refers to the first item added.
✏️
Mutable
You can add, remove, or change elements after the list is created. No need to rebuild it.
🎨
Heterogeneous
A single list can hold integers, strings, floats, booleans, even other lists — all at once.
↔️
Dynamic
The size changes automatically as you add or remove items. No fixed capacity to worry about.
Zero-indexed: Python counts from 0, not 1. The first element is always at index 0, the last at index -1.

04 Methods

List Methods

Methods are called with dot notation and mostly modify the list in place — meaning they change the original list and return None.

.append(x)
Add x to the end
lst.append(5)
.extend(iterable)
Add all items from iterable
lst.extend([6, 7, 8])
.insert(i, x)
Insert x before position i
lst.insert(0, "first")
.remove(x)
Remove first occurrence of x
lst.remove("apple")
.pop(i)
Remove & return element at i
lst.pop() # last item
.sort()
Sort the list in place
lst.sort(reverse=True)
.reverse()
Reverse the list in place
lst.reverse()
.index(x)
Return index of first x
lst.index("banana")
.count(x)
Count occurrences of x
lst.count(3)
.clear()
Remove all elements
lst.clear() # → []
.copy()
Return a shallow copy
new = lst.copy()
del lst[i]
Delete element at index i

05 Comprehensions

List Comprehensions

A concise way to build a list from any iterable. Reads like plain English: "give me X for each item in Y, if condition."

[  expression  for  item  in  iterable  if condition  ]
expression — what to keep / transform
item — loop variable
iterable — source to loop over
condition — optional filter
Common patterns
1# Squares of 1–5
2squares = [x**2 for x in range(1, 6)]
3# → [1, 4, 9, 16, 25]
4 
5# Even numbers only
6evens = [x for x in range(10) if x % 2 == 0]
7# → [0, 2, 4, 6, 8]
8 
9# Transform strings
10upper = [s.upper() for s in ["a", "b", "c"]]
11# → ["A", "B", "C"]
